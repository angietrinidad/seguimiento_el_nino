Esta página individualiza los servicios de salud y centros educativos que caen dentro de las zonas de riesgo de inundación, para planificación operativa (evacuación, refugios, continuidad de servicios).

  • No está enlazada en la navegación pública del sitio.

::: {.callout-warning}
## Uso administrativo — leer antes
Esta página **individualiza** los servicios de salud y centros educativos que
caen dentro de las zonas de riesgo de inundación, para planificación operativa
(evacuación, refugios, continuidad de servicios).

- **No está enlazada** en la navegación pública del sitio.
- **No es una página privada:** se publica en GitHub Pages, que es **público**;
  cualquiera con esta URL puede verla. Para privacidad real, esta página debe
  dejar de publicarse (generarse solo en local) o alojarse en un repositorio con
  control de acceso (ver nota al pie).
- **Educación: registro OFICIAL del MEC** (datos.mec.gov.py, período 2014),
  georreferenciado y con **matrícula** (cantidad de alumnos) por establecimiento.
- **Salud: registro OFICIAL del MSPBS/DIGIES** (establecimientos georreferenciados),
  con categoría oficial (USF, puesto, centro, hospital). No incluye camas.

Se regenera automáticamente con cada actualización de los datos (fecha arriba).
:::

```{python}
#| label: tablas-expuestos
#| echo: false
import json
from IPython.display import Markdown

feats = json.load(open("datos/geo/expuestos-inundacion.geojson", encoding="utf-8"))["features"]

def loc(lat, lon):
    u = f"https://www.openstreetmap.org/?mlat={lat}&mlon={lon}#map=17/{lat}/{lon}"
    return f"[{lat:.5f}, {lon:.5f}]({u})"

zonas = {}
for f in feats:
    p = f["properties"]
    lon, lat = f["geometry"]["coordinates"]
    p["_lat"], p["_lon"] = lat, lon
    zonas.setdefault(p.get("zona", "—"), {"salud": [], "educacion": []})
    zonas[p["zona"]][p["clase"]].append(p)

def tabla_salud(items):
    if not items:
        return "_Ninguno detectado en esta zona._\n"
    head = "| # | Establecimiento (MSPBS) | Categoría | Subtipo | Distrito | Ubicación |\n|---:|---|---|---|---|---|"
    filas = []
    for i, p in enumerate(sorted(items, key=lambda x: x.get("nombre") or ""), 1):
        cat = p.get("categoria") or p.get("tipo") or "—"
        filas.append(f"| {i} | {p.get('nombre') or '(sin nombre)'} | {cat} | {p.get('subtipo') or '—'} | {p.get('distrito') or '—'} | {loc(p['_lat'], p['_lon'])} |")
    return head + "\n" + "\n".join(filas)

def tabla_educacion(items):
    if not items:
        return "_Ninguno detectado en esta zona._\n"
    head = "| # | Institución (MEC) | Matrícula (año) | Distrito / barrio | Ubicación |\n|---:|---|---:|---|---|"
    filas = []
    for i, p in enumerate(sorted(items, key=lambda x: x.get("nombre") or ""), 1):
        mat = f"{p['matricula']} ({p.get('matricula_anio','')})" if p.get("matricula") is not None else "s/d"
        ubic = " / ".join(x for x in [p.get("distrito"), p.get("barrio")] if x) or "—"
        filas.append(f"| {i} | {p.get('nombre') or '—'} | {mat} | {ubic} | {loc(p['_lat'], p['_lon'])} |")
    return head + "\n" + "\n".join(filas)

out = []
tot_s = tot_e = tot_mat = 0
for zona in sorted(zonas):
    s = zonas[zona]["salud"]; e = zonas[zona]["educacion"]
    tot_s += len(s); tot_e += len(e)
    tot_mat += sum(p.get("matricula", 0) or 0 for p in e)
    out.append(f"## {zona}\n")
    out.append(f"**Servicios de salud expuestos: {len(s)}**\n")
    out.append(tabla_salud(s) + "\n")
    out.append(f"**Centros educativos expuestos: {len(e)}** "
               f"(matrícula sumada: {sum(p.get('matricula', 0) or 0 for p in e):,})\n")
    out.append(tabla_educacion(e) + "\n")

resumen = (f"**Total: {tot_s} servicios de salud y {tot_e} centros educativos** en zonas "
           f"de riesgo de inundación, con **{tot_mat:,} alumnos** matriculados (MEC) en las "
           f"escuelas expuestas.\n")
Markdown(resumen + "\n" + "\n".join(out))
```
