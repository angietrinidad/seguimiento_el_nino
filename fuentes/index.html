Inventario de fuentes consultadas y priorizadas, clasificadas por tipo y nivel
de confiabilidad. Se genera desde `fuentes/registro.csv`.

```{python}
#| echo: false
import csv
from IPython.display import Markdown

rows = list(csv.DictReader(open("fuentes/registro.csv", encoding="utf-8")))
head = "| Fuente | Tipo | Confiabilidad | Nivel | Enlace |\n|---|---|---|---|---|"
def link(r):
    return f"[{r['nombre']}]({r['url']})" if r['url'] else r['nombre']
body = "\n".join(
    f"| {link(r)} | {r['tipo']} | {r['confiabilidad']} | {r['nivel_geografico']} | [↗]({r['url']}) |"
    for r in rows
)
Markdown(head + "\n" + body)
```
